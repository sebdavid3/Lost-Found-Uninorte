## Identificacion de actores

| Actor | Tipo | Objetivo principal |
| :--- | :--- | :--- |
| Estudiantes y colaboradores | Primario | Reportar objetos encontrados y buscar pertenencias perdidas |
| Administradores (Bienestar) | Primario | Validar propiedad de objetos, gestionar inventario y realizar entregas |
| Personas externas | Secundario | Reportar hallazgos accidentales |
| Area de seguridad | Secundario | Custodia fisica de los objetos encontrados |
| Depto. de tecnologia | Secundario | Soporte tecnico y mantenimiento del sistema |
| Directivos | Secundario | Definir politicas de gestion y consultar indicadores |

## Responsabilidades por actor

| Actor | Responsabilidades |
| :--- | :--- |
| Estudiantes y colaboradores | Reportar hallazgos, consultar catalogo y enviar solicitudes de reclamo con evidencias |
| Administradores (Bienestar) | Crear/editar/eliminar registros, validar pruebas de propiedad y cerrar casos |
| Personas externas | Reportar hallazgos y solicitar informacion basica de contacto |
| Area de seguridad | Custodiar objetos y coordinar ubicacion fisica con Bienestar |
| Depto. de tecnologia | Monitorear plataforma, base de datos y resolver incidencias |
| Directivos | Revisar estadisticas y definir lineamientos de gestion |

## Interaccion con el sistema

- Estudiantes y colaboradores: consultan catalogo publico, filtran por categorias y envian solicitudes de reclamo con evidencias.
- Administradores (Bienestar): operan panel administrativo, validan propiedad, gestionan inventario y cierran casos.
- Personas externas: usan formularios de reporte y canales de contacto basico.
- Area de seguridad: verifica correspondencia entre registro digital y ubicacion fisica.
- Depto. de tecnologia: administra soporte tecnico y disponibilidad del sistema.
- Directivos: consumen reportes y estadisticas para decisiones de gestion.
